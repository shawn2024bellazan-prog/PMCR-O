## What This Is Not

PMCRO is not a prompt pattern. Prompt patterns optimize inputs to a fixed system. The system that runs these cycles is not fixed — it deepens with each cycle, accumulating constraints, refining its self-model, growing more capable in its domain without retraining.

PMCRO is not fine-tuning. Fine-tuning adjusts weights in a separate pass, after the fact. The constraint accumulation here happens at inference time, inside the cycle, as the loop runs.

PMCRO is not retrieval. LockedThoughts are not stored answers retrieved on demand. They are crystallized self-knowledge that seeds new generation. The loop runs on them. It does not fetch from them.
